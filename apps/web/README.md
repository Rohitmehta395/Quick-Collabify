# Web Frontend

The primary user-facing frontend for the collaborative workspace application, built using Next.js, React, and Tailwind CSS.

## Responsibilities

- Render the UI for user management, document browsing, and the collaborative editor.
- Connect to the `api` service for REST requests and the `realtime` service via WebSockets for live collaboration.

## Commands

- `pnpm dev`: Start the Next.js development server.
- `pnpm build`: Build the production optimized Next.js bundle.
- `pnpm start`: Start the production Next.js server.
